# 🤖 NayePankh AI Chatbot

An AI-powered chatbot developed for the AI Internship Selection Task of NayePankh Foundation.

The chatbot helps students by answering questions related to:

* Career Guidance
* Internships
* Programming
* Technology
* Skills Development
* Education

It uses the Groq API with the **Llama 3.3 70B Versatile** model to generate fast and intelligent responses.

---

## 🚀 Features

* AI-powered conversational chatbot
* Chat history using Flask sessions
* Powered by Groq API
* Uses Llama 3.3 70B Versatile model

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a `.env` file

```env
GROQ_API_KEY=your_groq_api_key
```

4. Run the application

```bash
python app.py
```

5. Open your browser

```text
http://127.0.0.1:5000
```
